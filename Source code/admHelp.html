How account creation works?

The admin only have the exclusive authority to create accounts for users. This process involves setting up new accounts and entering the necessary user details. Additionally, the admin can modify user details and delete accounts as needed. This centralized control allows the admin to manage user access and ensure that only authorized individuals have access to the system.

How do I restore deleted users?

Deleted users cannot be restored, whether the deletion was accidental or intentional. If a user needs to regain access, you will need to create a new account for them. This means that the admin have to re-enter their details and set up their access.

How do I change my password?

To change your password, navigate to the Account and Settings section and find Security. Here, you can update your password. It is recommended to create a password that is at least 8 characters long and includes a combination of uppercase and lowercase letters, numbers, and special symbols. This helps ensure your account remains secure.

Why the consumption does not update everytime I refresh the page?

If your total consumption does not update immediately when you refresh the page, there might be temporary server issues. While these issues are being resolved, rest assured that your consumption is still being accurately recorded. The data will appear correctly on your interface once the issue is fixed.

How do I retrieve the previous total consumption of my apartment?

Previous consumption data is not accessible through the system. The system retains only the consumption records from the current and the most recent for viewing. For records, you may need to check with your utility provider for any possible archived data.
